
# Nature Ball

Nature Ball is a Mario-style platformer where you play as Nico the Monkey using a banana revolver to shoot tree balls, defeat spiders, and save your kidnapped wife Nica.

## Controls
- W = Jump
- A = Move Left
- D = Move Right
- S = Drop/Crouch
- Space = Shoot

## Setup
1. Open `index.html` in a browser
2. Play the game
